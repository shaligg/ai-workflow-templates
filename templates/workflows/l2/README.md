# L2 Feature Task Template

Default workflow for feature development.

Flow:

1. Create feature docs (recommended):
   `bash .workflow/scripts/create-feature.sh <name>`
2. Fill `requirements.md` (`Raw Input` optional).
3. Create `design.md`.
4. Create `tasks.md`.
5. Default: implement next unfinished task.
6. Explicit user request can switch to complete-all mode.
