# L2 Feature Task Template

Use this template for normal feature development.

Flow:

1. Feature request
2. Paste raw product/planning text into `requirements.md` (`Raw Input`)
3. Extract and normalize `requirements.md`
4. Design document (`design.md`)
5. Task breakdown (`tasks.md`)
6. Default: implement next unfinished task (explicit user request can switch to complete-all mode)
7. Review
